# lesson

A durable rule or convention, plus the cost that earned it. Default `status: confirmed`.

Worth capturing - include what applies, and anything else that matters:

- the rule, stated plainly ("always X", "never Y")
- the cost that earned it - the incident, the wasted hours, the outage
- where it applies, and where it does not

A rule with no cost is unmotivated and gets ignored, so the cost is the part that makes it
stick. If it refines an earlier lesson, add `supersedes`.
