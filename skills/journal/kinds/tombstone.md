# tombstone

Marks non-existence: an approach tried and abandoned, or work deliberately not done. Here lies
the path not taken, kept so no one walks it again. Always `status: confirmed`.

Worth capturing - include what applies, and anything else that matters:

- what was abandoned or never built
- why it was killed or skipped
- what is done instead, if anything

Code cannot show what is absent; this entry is the only trace. If it buries an earlier `draft`
or `decision`, add `supersedes`; if it closes a `reminder`, add `resolves`.
