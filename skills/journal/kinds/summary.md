# summary

A session recap that consolidates many commits into one thread and names what is still open.
Default `status: confirmed`.

Worth capturing - include what applies, and anything else that matters:

- what changed and why it mattered - the narrative the git log cannot carry, not a
  commit-by-commit replay
- the surprises and the reasoning
- if a hard bug was cracked: its root cause and the hypotheses that were tried and failed
- what is still open, so the next session can pick up

If it continues a prior session, add `follows-up`.
